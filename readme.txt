Installation:

1. Copy the extracted files to your web root directory
2. Change the server name,username and password in model/Model.php on line 13
3. By default the database name is set, If needed change the database name in model/Model.php on line 14
4. By default the table name is set,If needed change the table name in model/Model.php on line 15
5. Run HypeSetup.php from the root directory, if the root is http://localhost then run http://localhost/HypeSetup.php
6. Make sure the configured database name(Ref.3) and table name(Ref.4) are created
7. Run the application from the root directory, if the root is http://localhost then run http://localhost/index.php
